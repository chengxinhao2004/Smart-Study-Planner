# Smart-Study-Planner
Project Overview: Presentation and system demonstration of the "Smart Study Planner," a Java-based desktop application designed to help students manage academic deadlines through intelligent urgency ranking and secure data privacy. ## How to Run
1. Install **XAMPP** and import the `project_db.sql` file provided in the `/database` folder.
2. Open the project in your IDE (e.g., Eclipse).
3. Ensure your database connection settings in `DBConnection.java` match your local XAMPP settings.
4. Run `Main.java`.

## Features
- Multi-user authentication with data isolation.
- Pro-style report generation with automated urgency ranking.
- Integrated dashboard with performance analytics.
